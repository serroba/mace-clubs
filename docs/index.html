Built for the swing.

No crowded dashboards. Just the cues and controls that matter when both hands are on the mace: tempo, intervals, sets, and a clean Garmin activity when you finish.

01

Feel the tempo

A drift-resistant 20–240 BPM metronome with vibration, tone, and configurable accents.

02

Follow the session

Built-in and custom work/rest plans call each transition and count completed sets.

03

Train eyes-free

Physical-button controls and wrist cues keep attention on the movement, not the screen.

04

Keep the record

Save heart rate, training load, battery use, and set count to a named Garmin activity.

Mace & Clubs is MIT-licensed and built in the open. Download a release or compile it with the Garmin Connect IQ SDK for the Instinct 3 Solar 45 mm.

git clone https://github.com/serroba/mace-clubs.git
cd mace-clubs

make build

# with the Connect IQ simulator running
monkeydo bin/mace-clubs.prg instinct3solar45mm
